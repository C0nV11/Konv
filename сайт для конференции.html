CSS язык. Значение языка в визуализации и структурировании веб-сайтов

CSS язык используется для блочного структурирования, верстки и визуализации HTML веб-страниц. В сравнении с возможностями визуализации HTML, CSS имеет больший спектр функций, а так же за счет свойства блочного отношения к тегам HTML, делает кодировку более удобной, а сам код менее грамоздким.
CSS позволяет создавать анимированные объекты а так же способен придать какой либо отклик объекта при взаимодействии с ним, к примеру: реакция кнопки на наведение на нее курсора, анимированное открытие меню сайта, анимированная загрузка сайта и многое другое.
Одним из простейших но не менее важным в создании сайта является кнопка, благодаря которой выполняются основные функции многих сайтов, но не смотря на всю важность программирование кнопки не является чем-то трудным, собственно сейчас мы с вами и напишем кнопку и даже анимируем ее.

Создание кнопки благодаря CSS

HTML:
<a href="ссылка"><button class="btn">Ваша кнопка</button></a><br>
<a href="ссылка"><button class="btn">Ваша кнопка 2</button></a>


CSS:
.btn {
width: 100px;
height: 50px;
background: #ff4500;
cursor: pointer;
align-items: center;
font-size: 17px;
font-weight: 500;
}

Рассмотрим каждый CSS элемент по отдельности:
width - отвечает за ширину программируемого элемента HTML страницы;
height - при помощи него изменяется высота элемента;
cursor с атрибутом pointer изменяет курсо из "стрелочки" в "указатель";
align-itemsзадает положение программируемого обьекта на веб-странице;
font задает шрифт текста на веб-странице или шрифт текста на самом программируемом элементе (в нашем случае это кнопка);
Атрибут тега font, size задает размер шрифта;
Атрибут weight задает олщину шрифта;
background может относиться как к элементам страницы так и к самому веб-сайту, используется для именения цвета элемента, фона элемента или самой страницы, цвет указывается благодаря базовым атрибутам HTML таким как red, blue, green, для более точного указания цвета используются адаптивные цветовые модели для кодирования цвета. Самыми распространенными являются RGB форма записи такова: background: rgb( 100, 100, 100);, RGBA: background: rgba(100,100,100);, HEX: background: #fff;

Однако не смотря на свойство блочного взаимодействия с элементами веб-страницы, CSS так же может ображаться к тегам поотдельности.
Предлагаю взглянуть на элемент class находящийся внутри тэга <button>.
Как вы можете заметить в кавычках написано сокращение btn от английского button, однако так делают в основном при работе нескольких человек над одним проектом для всеобщего удобства, в противном случае там может быть написано все что угодно, главное это продублировать это в CSS и перед началом элемента, который Вы предварительно обозначили блягодаря элементу class, поставить точку.
Рассмотрим это на примере приведенном ранее, но теперь вместо двух одинаковых названий мы напишем две разные кнопки которые будут по разному отображаться пользователю

HTML:
<a href="ссылка"><button class="btn">Ваша кнопка</button></a><br>
<a href="ссылка"><button class="button">Ваша кнопка 2</button></a>


CSS:
.btn {
width: 100px;
height: 50px;
background: #ff4500;
cursor: pointer;
align-items: center;
font-size: 17px;
font-weight: 500;
}
.button {
width: 125px;
height: 50px;
background: #20b2aa;
cursor: pointer;
align-items: center;
font-size: 20px;
font-weight: 700;
color: white;
}

Свойства CSS не заканчиваются на расскрашивании кнопок, так же, к примеру, эту же кнопку можно переместить с помощью таких компонентов кода как margin-top(по вертикали) и margin-left(по горизонтали)


HTML:
<a href="ссылка"><button class="btn">Ваша кнопка</button></a><br>
<a href="ссылка"><button class="button">Ваша кнопка 2</button></a>


CSS:
.btn {
width: 100px;
height: 50px;
background: #ff4500;
cursor: pointer;
align-items: center;
font-size: 17px;
font-weight: 500;
}
.button {
width: 125px;
height: 50px;
background: #20b2aa;
cursor: pointer;
align-items: center;
font-size: 20px;
font-weight: 700;
color: white;
margin-left: 100px;
margin top: -80px;
}

Создания анимации для программируемого объекта


Достаточно итенересной и разносторонней функцией CSS является анимации. Самый простой способ придать анимацию кнопке это тег hover. Он предназначен для того что бы при наведении курсора на кнопку она воспроизводила анимацию заданную через соответствующие атрибуты, такие как backgroud (изменяет фон при наведении курсора) width(анимерованно заполняет кнопку по горизонтали) height (анимированно заполняет кнопку по вертикали) animation заставляет обьект воспроизводить элемент определенную анимацию заданную соответствующими псевдо-элементами. Добавим горизонтальное заполнение кнопки при наведении.

Для удобства напишем новую кнопку

HTML:
<a href="ссылка"><button class="btm">Ваша кнопка</button></a><br>

CSS:
.btm {
border: .25em solid black;
padding: 1em 3em;
color: black;
background: linear-gradient( to top left, pink, powderblue);
border-radius: 1em;
position: relative;
transition: all 0.3s;
}
.btm::after {
content: "";
position: absolute
; top: 120%;
left: 0;
height: 100%;
width: 100%;
}
.btm:hover {
color: gray;
filter: blur(1px);
background: linear-gradient(to top left, pink, powderblue) ;
box-shadow: black;
box-shadow: 0 0 5px,
0 0 20px,
0 0 50px,
0 0 100px;
}

Назовем будущую кноку как btn. Далее добавим рамку для кнопки border. Через padding можно изменить положение как кнопки так и элементов находящихся внутри нее. С помощью color добавим свой цвет рамке. background-color изменит цвет и через его элемент linear-gradient этот фон сделаем градиентовым.
Следом скруглим углы рамки кнопки, с этим нам поможет элемент border-radius, сила "скругления" углов изменяется как и ширина рамки border-radius: 1em. Для внесения именений используют или "число"em или "число"px
transition изменяет скорость анимации. При работе с анимациями программирование кнопки нужно разделить ее программирование на несколько частей.
Первой частью кнопки является ее внешний вид.
Второй частью является позиционирование анимации на самой кнопке и размер анимации относительно кнопки, после ее CSS-названия требуется прописать псевдо-элемент ::after.
Третья часть это ее внешний вид, где после названия кнопки прописывается :hover.
Элемент color изменит цвет текста кнопки после наведения на нее курсора.
Элемент filter добавит фильтр кнопки после наведения на нее, в нашем случае благодаря элементу filer: blur при наведении кнопка будет размываться. Как в случае с рамкой сила действия элемента filter изменяется через em или px
ПРИМЕЧАНИЕ: лучше использовать при изменении силы действия элемента px для большей точности, так же при изменении силы действия элемента можно использовать десятичные дроби для достижения максимальной точности.


Элемент background изменит цвет кнопки при наведении на нее, если прописакть другой цвет.
При добавлении элемента box-shadow можно добавить анимацию появления тени при наведении на кнопку, это даст особенный эффект при работе с 3D анимацией элемента, для изменения размера тени требуется прописать второй раз элемент box-shadow где вместо цвета указать требуемый размер.
